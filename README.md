# ECモール商品分析ダッシュボード

楽天・Yahoo!ショッピング・Amazonの商品データを統合管理するWebアプリケーションです。

## 🌟 主な機能

- **CSVデータのアップロード**: 各モールのCSVファイルを簡単にアップロード
- **スプレッドシート連携**: Googleスプレッドシートに自動保存・読み込み
- **日次推移分析**: アクセス数、売上、CVRなどの推移をグラフで可視化
- **施策記録**: 実施した施策を記録し、効果を測定
- **効果測定**: 施策前後7日間のデータを比較して効果を数値化
- **ブラウザ保存**: 設定はブラウザに保存され、毎回入力不要
